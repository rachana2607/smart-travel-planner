# 🌍 Smart Travel Planner

An AI-powered travel planning application that helps users generate a complete travel itinerary based on their destination and travel dates.

The system collects weather data, tourist places, and hotel information and then uses an AI model to generate a personalized travel plan.

---

## 🚀 Features

- 🌤 Get real-time **weather information**
- 📍 Discover **top tourist places**
- 🏨 Find **available hotels**
- 🧠 Generate **AI-powered travel itinerary**
- 🗓 Automatically adjusts itinerary based on **trip duration**
- 🎨 Interactive UI with background styling

---

## 🛠 Tech Stack

- **Python**
- **Streamlit**
- **LangChain**
- **Groq LLM**
- **OpenWeatherMap API**
- **Travel APIs**
- **HTML / CSS**

  ## ⚙️ How It Works

1. User enters **city name**
2. User selects **start date and end date**
3. The system fetches:
   - Weather information
   - Tourist places
   - Hotel recommendations
4. The AI generates a **day-wise travel itinerary**
